Social Presidents

To create these 2 graphs seen above, I was able to scrape data from Twitter's API and give each of the last 1600 tweets a sentiment score based upon words that were used in their Tweets. These graphs were then created to compare each of the Presidental candidates' showing of emotion when Tweeting. This involved grabbing data, cleaning the data, joining the data, and then comparing the 2 sets of data in order to produce these graphs.
